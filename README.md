# CCTV Correction Pro 📹

A high-performance Desktop Application built with **Electron.js** and **Node.js** to streamline attendance correction workflows. This tool was developed to handle real-world security data management at **Diamond Clothing Industries**.

## 🚀 Top Features
* **Persistent Storage**: Full CRUD operations powered by an integrated **SQLite3** database.
* **Dual Mode UI**: Seamless toggle between professional Dark and Light themes for any office lighting.
* **Workflow Shortcuts**: High-speed entry using `Ctrl + S` (Save), `Ctrl + P` (Print), and `Ctrl + N` (New).
* **Data Integrity**: Built-in date restrictions (2026) and input validation to prevent logging errors.
* **Print-Ready Reports**: Custom filtered views designed for professional PDF generation.

## 🛠️ Tech Stack
- **Runtime**: Node.js & Electron
- **Database**: SQLite3
- **Styling**: CSS3 (Glassmorphism & Responsive Design)
- **Icons**: Font Awesome 6.0

## 📦 Local Installation
1. Clone the repository.
2. Run `npm install` to get dependencies.
3. Run `npx electron-rebuild` to sync the database with your Electron version.
4. Launch with `npm start`.
